Tema 2

Grundlæggende web

Læring

I dette tema fik jeg en grundlæggende forståelse for HTML og CSS. Jeg arbejdede med opbygning og struktur i HTML samt med farver, typografi og layout i CSS. Her lærte jeg også om grid, flexbox og media queries med fokus på responsivt design og en mobile-first-tilgang. Jeg lærte om visuelt design og centrale designprincipper, herunder designkonventioner og gestaltprincipper og fik kendskab til opsætning af tekst og billeder i Figma. Jeg lærte at arbejde med billedbehandling og webformater som WebP og SVG. Inden for kodning lærte jeg også om semantisk markup, mappestruktur og validering af HTML og CSS. Afslutningsvis fik jeg kendskab til, hvordan man uploader projekter til GitHub.

Projekt

Temaets projekt var en studiestartsprøve, hvor jeg udarbejdede et reponsivt website på basis af udleveret tekst, billeder, layoutdiagram og wireframe. Jeg opsatte billeder og tekst i HTML, og stylede med fonte, farver, margin, grid og flexbox i CSS.

Process og løsning

Undervejs i processen oplevede jeg flere udfordringer, da både HTML og CSS var nyt for mig, og jeg skulle forstå sammenhængen mellem de to. Særligt media queries var udfordrende, og det tog tid at opnå en forståelse for formål og anvendelse. Jeg kom i mål med hjemmesiden, og jeg så tydeligt, hvor meget jeg havde lært og udviklet mig gennem temaet.
